# Personal game preservation
Having many CDs and DVDs with games from the old time, I try to preserve the files of those games to lower the chances of loosing them. I'm also not fancy buying them, cause they're pretty pricey. Secondly, I also try to run each of them to make sure they're playable and at the same time I try to experimennt with playing games on Linux, which is sometimes the only option through stacked incompatibilities of modern Windows and because of high risk of bricking the system with certain types of DRMs.

This repo is serving the purpose of documenting my work for future reference and to help others dealing with similar task.

> [!IMPORTANT]
> This is not intended for Piracy but for users to exercise their Fair Use rights for the games they actually own and can no longer play due to draconian and unsupported copy protection
